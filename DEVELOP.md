## Setup

clone this repository

```shell
git clone git@github.com:porco-rosso-j/aztec-mobile-wallet-dry.git
cd aztec-mobile-wallet-dry
```

move to this branch

```shell
git fetch tags
git checkout main
```

clone submodules

```shell
 git submodule update --init --remote
```
